### iTerm - Shell

#### iTerm快捷键

```
进入设置：command + ,
快速唤出：进入设置 -> Keys
垂直分割：command + d
水平分割：command + shift + d
删除一个单词：ctrl + w
删除当前行：ctrl + u
清屏：ctrl + l
双击：选中当前单词（选中即复制）
三击：选中当前行（选中即复制）
```
